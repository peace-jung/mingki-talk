# mingki-talk Server

## Skill

- Node.js (Server)

- PostgreSQL (RDB)

- Heroku (Host)

## Install

```
npm install (or use yarn)

npm start
```

If you use database, you should set <code>/.env</code> file.

```
DATABASE_URL=[your url]
PORT=[your port] (default port: 3000)
```
